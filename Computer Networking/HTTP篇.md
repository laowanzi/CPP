## HTTP基本概念

***

### HTTP是什么？

HTTP是超文本传输协议，也就是HyperText Transfer Protocol

超文本：超越了普通文本的文本，超文本中有其他文本的链接

### HTTP常见的状态码有哪些？

| 状态码分类 | 具体含义   | 常见的状态码 |
| ---------- | ---------- | ------------ |
| 1xx        | 提示信息   |              |
| 2xx        | 成功       |              |
| 3xx        | 重定向     |              |
| 4xx        | 客户端错误 |              |
| 5xx        | 服务器错误 |              |
