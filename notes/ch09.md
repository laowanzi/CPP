#### 顺序容器概述

***

| 容器名称       | 概述         |
| -------------- | :----------- |
| `vector`       | 可变大小数组 |
| `deque`        | 双端队列     |
| `list`         | 双向链表     |
| `forward_list` | 单向链表     |
| `array`        | 固定大小数组 |
| `string`       | 字符串       |

#### 迭代器

***

满足以下条件，两个迭代器`begin`和`end`构成一个迭代器范围：

* 指向同一个容器中的元素，或者容器最后一个元素之后的位置
* `end`不在`begin`之前，当`begin=end`时，容器为空
